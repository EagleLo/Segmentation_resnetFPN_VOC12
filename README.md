## Semantic Segmentation

Semantic segmentation implementation using pytorch framework.

### Currnet
- FCN8
- U-Net
- PSPNet

### Dependencies
* [Python 2.7 or 3.5+](https://www.continuum.io/downloads)
* [PyTorch 0.4.0](http://pytorch.org/)
